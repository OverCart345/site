Загрузка файлов на сервер


<form method="post" enctype="multipart/form-data">
<input type="file" name="unicalyName111">
<input type="submit">
</form>

<?php

/*
 ЗАГРУЗКА ФАЙЛОВ НА СЕРВЕР

 HTML
 1. Создаём форму с атрибутами method="post" enctype="multipart/form-data"
 2. Создать поле ввода с type="file" и уникальным именем name=""

 PHP
 3. Проверить с помощью if и isset($_FILES["ИМЯ_ПОЛЯ_ВВОДА"]) наличие поля ввода
 4. Записать в переменную путь куда будет сохраняться изображение. ВНУТРИ УСЛОВИЯ
    ($path = 'upload/'.$_FILES['ИМЯ_ПОЛЯ_ВВОДА']['name'];  Получим upload/myImage.png)
 5. Перемещаем наш файл по пути, который лежит в переменной. ВНУТРИ УСЛОВИЯ
    (move_uploaded_file($_FILES['ИМЯ_ПОЛЯ_ВВОДА']['tmp_name'], $path);)


*/
$path = "";
if(isset($_FILES["unicalyName111"])){
  $path = 'upload/'.$_FILES['unicalyName111']['name'];  // upload/myImage.png
  move_uploaded_file($_FILES['unicalyName111']['tmp_name'], $path);
}

?>

SQL-запросы


/* SQL-ЗАПРОСЫ

0. Создать подключение
    - $conn = new mysqli($host, $user, $password, $dbname);

1. СОЗДАНИЕ ЗАПРОСА
   - Нужно создать переменную, в которую вы положите запрос

1.1 ДОБАВЛЕНИЕ
$sql_query = 'INSERT INTO `Tovar`(`Name`, `price`) VALUES ("ЗНАЧЕНИЕ1","ЗНАЧЕНИЕ2")';
         ВСТАВИТЬ строку В таблицу Tovar, в столбцы Name и price, ЗНАЧЕНИЯ "ЗНАЧЕНИЕ1" и "ЗНАЧЕНИЕ2"

1.2 ИЗМЕНЕНИЕ
$sql_query = 'UPDATE `Tovar` SET `Name`="ЗНАЧЕНИЕ1",`price`="ЗНАЧЕНИЕ2" WHERE name="STEPAN"';
            ИЗМЕНИТЬ строку в таблице Tovar, установить Name=ЗНАЧЕНИЕ1 и price=ЗНАЧЕНИЕ2, ГДЕ name=STAPAN

1.3 УДАЛЕНИЕ
$sql_query = 'DELETE FROM `Tovar` WHERE name="STEPAN"';
            УДАЛИТЬ строку ИЗ таблицы Tovar, где name="STEPAN"

1.4 ПОЛУЧЕНИЕ
$sql_query = 'SELECT `id`, `Name`, `price` FROM `Tovar`';
            ПОЛУЧИТЬ все id, Name и price из таблицы Tovar

1.5 ПОДСТАНОВКА ПЕРЕМЕННЫХ
Во все запросы можно подставлять переменные с данными, например при добавлении:
$sql_query = 'INSERT INTO `Tovar`(`Name`, `price`) VALUES ($ИМЯ_ПЕРЕМЕННОЙ_1, $ИМЯ_ПЕРЕМЕННОЙ_2)';
          ВСТАВИТЬ строку В таблицу Tovar, в столбцы Name и price, ЗНАЧЕНИЯ "ТО ЧТО ЛЕЖИТ В ИМЯ_ПЕРЕМЕННОЙ_1" и "ТО ЧТО ЛЕЖИТ В ИМЯ_ПЕРЕМЕННОЙ_2"

2. Отправить запрос
$result = $conn->query($sql_query);

2.1 Если использовали INSERT или UPDATE или DELETE
    - в $result будет лежать true или false. 
    Далее с ней ничего особо делать не надо
    
2.2 Если использовали SELECT
      $sql_query = 'SELECT `id`, `Name`, `price` FROM `Tovar`';
      $result = $conn->query($sql_query);
    - в $result будет лежать массив всех строк

    Получение данных:
*/
while ($row = $result->fetch_assoc()): ?>

    <div class='lalala'>
      <h1 class='lalala2'> <php echo row["Name"]; ?> </h1>
      <p class='lalala3'> <php echo row["price"];  ?> </p>
    </div>

<?php endwhile; ?>
